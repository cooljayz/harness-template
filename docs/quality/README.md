# Quality Grades

기능별 품질 등급과 기준을 둔다. 에이전트는 타겟 등급에 맞는 센서(테스트/관측)를 구성해야 한다.

## 등급

| 등급 | 의미 | 요구 사항 |
|---|---|---|
| **A** | 사용자에게 직접 영향 (결제, 인증) | 단위 + 통합 + E2E + 뮤테이션 테스트, SLO 지표, 장애 플레이북 |
| **B** | 내부 핵심 경로 | 단위 + 통합, 커버리지 ≥ 80%, 기본 로깅 |
| **C** | 보조 도구/스크립트 | 단위 테스트, 타입 체크 |
| **D** | 실험/스파이크 | 최소한 빌드는 성공 |
